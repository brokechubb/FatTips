# Privacy Policy

**FatTips by CodeStats**
**Effective Date:** March 24, 2026
**Last Updated:** March 24, 2026

This policy explains what data we collect, why, and how we handle it.

---

## 1. Who We Are

FatTips is operated by CodeStats ("we", "us", "our"), based in the United States. Contact us at info@codestats.gg.

## 2. Data We Collect

### Data You Provide

| Data                         | Why We Store It                            |
| ---------------------------- | ------------------------------------------ |
| Discord User ID              | Identifies your account and wallet         |
| Solana wallet public key     | Links your on-chain wallet to your account |
| Encrypted private key + salt | Allows you to transact through the bot     |
| API keys (if created)        | Authenticates programmatic access          |

### Data Generated by Usage

| Data                     | Why We Store It                                            |
| ------------------------ | ---------------------------------------------------------- |
| Transaction history      | Tips, sends, withdrawals, swaps -- records of all activity |
| Airdrop records          | Airdrop creation, claims, settlement status                |
| Guild (server) settings  | Per-server configuration (prefix, channels)                |
| Activity timestamps      | Tracks recent channel activity for rain/airdrop targeting  |
| API key usage timestamps | Tracks last use for security monitoring                    |

### Data We Do NOT Collect

- Email addresses
- IP addresses
- Discord messages or message content (beyond command inputs)
- Browser fingerprints or tracking cookies
- Unencrypted private keys (we cannot decrypt them without the master key)

## 3. How Data Is Stored

- **Database:** PostgreSQL, self-hosted on our infrastructure.
- **Encryption:** Private keys are encrypted with AES-256-GCM using a unique salt per user. The master encryption key is stored as an environment variable, never in the database.
- **Cache:** Redis is used for temporary activity tracking and job queues. Activity data expires automatically.
- **Backups:** Database backups are created during deployments. Backups are stored on our server.

All infrastructure runs on our own server at codestats.gg. We do not use third-party cloud database providers.

## 4. Blockchain Data

Solana is a public blockchain. When you send a tip, create an airdrop, or make any on-chain transaction:

- The transaction is **publicly visible** on the Solana blockchain forever.
- Your wallet address, amounts, and timestamps are on-chain and cannot be deleted.
- Anyone can view this data using a block explorer (e.g., Solscan).

We have no control over on-chain data. This is inherent to how blockchains work.

## 5. Third-Party Services

We interact with the following external services:

| Service             | What For                | Data Shared                                          |
| ------------------- | ----------------------- | ---------------------------------------------------- |
| Discord API         | Bot functionality       | Discord User ID, command interactions                |
| Solana RPC (Helius) | Blockchain transactions | Wallet addresses, transaction data                   |
| Jupiter API         | Token pricing and swaps | Token amounts (no user identity)                     |
| Sentry              | Error tracking          | Error logs (may include Discord IDs in stack traces) |

We do not sell, rent, or trade your data to any third party.

## 6. Data Retention

- **Account data** (Discord ID, wallet) is retained as long as your account exists.
- **Transaction history** is retained indefinitely for record-keeping.
- **Activity tracking data** in Redis expires automatically (typically within minutes).
- **API keys** are retained until revoked. Revoked keys are soft-deleted (marked inactive).

## 7. Your Rights

You can:

- **Export your private key** at any time using `/wallet export-key` and take full custody of your wallet.
- **Request your data** by contacting info@codestats.gg.
- **Request deletion** of your account data by contacting info@codestats.gg. Note: on-chain transaction history cannot be deleted (see Section 4).

If you export your key and stop using the Service, we retain your database records but your wallet operates independently of us.

## 8. Children

FatTips is not intended for anyone under 18. We do not knowingly collect data from minors. If you believe a minor is using the Service, contact us and we will remove their data.

## 9. Security

- Private keys are encrypted at rest with AES-256-GCM.
- Database and Redis are accessible only within our Docker network, not exposed to the public internet.
- API access requires authenticated keys tied to individual users.
- We use Sentry for real-time error monitoring.

No system is 100% secure. While we take reasonable measures to protect your data, we cannot guarantee absolute security. See our [Terms of Service](./TERMS_OF_SERVICE.md) for liability details.
